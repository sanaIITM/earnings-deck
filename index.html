-- Revenue breakdown query (sample)
WITH rev AS (
  SELECT region, product, SUM(amount) AS revenue
  FROM ledger
  WHERE quarter = '2025-Q2'
  GROUP BY region, product
)
SELECT region, SUM(revenue) AS total_revenue
FROM rev
GROUP BY region
ORDER BY total_revenue DESC;

Valuation & Return

Net Present Value (NPV):

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \]

Capital Asset Pricing Model (CAPM):

\[ E(R_i) = R_f + \beta_i\,(E(R_m) - R_f) \]

Use NPV for project decisions; CAPM informs discount rates. Keep r consistent with risk regime.
